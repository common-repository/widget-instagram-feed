== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/widget-instagram-feed` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the `'Widget Instagram feed'` screen in WordPress
1. Use the `Settings->Widget Instagram feed` to configure the plugin
